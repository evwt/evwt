# Tutorial

Writing your first EVWT app.

## Prerequisites

Make sure you've followed the quick start instructions in the README first, reproduced below:

```bash
npm install -g @vue/cli
vue create my-evwt-app
cd my-evwt-app
vue add electron-builder
npm install evwt
```
